The folders in this directory contains functions and programs for calculations of problems in Discrete Mathematics and is designed for use by any student studying the course.

Some files are programs that should be run as a whole while others contain functions; this is laid out in the READ ME's of the individual topic's folder.

It's expected that the user is familiar with python enough to run functions and insert valid parameters.
Some files contains auxiliary functions which sometimes can be used on their own but is intended for use by other functions.

The files have been tested and debugged with previous exams and assignments for the course.

The author disclaims any responsibility of errors that may lead to wrong answers.
